1. git remote add origin (link of github repo)
2. git remote -v ( to verify the remote exits )
3. git branch (to check branch)
4. git branch -m (new name)( to rename branch)
PS C:\Users\user\Github\localrepo> echo "# localrepo" >> README.md
>> git init
>> git add README.md
>> git commit -m "first commit"
>> git branch -M main
>> git remote add origin https://github.com/Ajinkya-commits/localrepo.git
>> git push -u origin main
